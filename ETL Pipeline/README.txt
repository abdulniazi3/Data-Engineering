# API Data Pipeline (Python)

This project demonstrates a simple ETL pipeline using Python.

## What it does

* Extracts data from a REST API
* Transforms JSON data into structured format
* Stores cleaned data for further use

## Technologies Used

* Python
* Jupyter Notebook
* REST APIs
* JSON

## Project File

* `data_pipeline.ipynb`

## Future Improvements

* Add database integration (PostgreSQL)
* Automate pipeline execution
